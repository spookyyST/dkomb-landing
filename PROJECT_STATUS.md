# dkomb.ru landing project notes

## Current stage

Static premium stub/landing for `dkomb.ru` is partially implemented. The page already has:

- full-screen loader with logo and frame-loading progress;
- fixed header with logo and anchor navigation;
- animated hero with mission text, frame sequence, and split showcase;
- manifest/company block with short thesis copy;
- media band with production/aviation imagery and overlay text;
- numbered directions block for activated carbon, charcoal production, and aviation parts;
- contact section with phone, email, addresses, and demo request form;
- SEO meta, Open Graph, and JSON-LD Organization;
- vanilla JS foundation without a build step.

## First brief coverage

Done:

- visual landing foundation that does not look like an empty placeholder;
- logo, base navigation, footer, contacts;
- modern industrial editorial style;
- form contract for future backend;
- structure suitable for later expansion.

Partially done:

- history/concept exists as a manifest, but not yet as a clear timeline;
- one-page anchor navigation exists, but not multi-page architecture;
- directions can become a catalog basis, but there is no satellite-site catalog yet;
- visual case/story slots are implied, but not implemented as a dedicated section;
- responsive CSS exists, but mobile/tablet visual QA is not complete.

Not done:

- dedicated stories/cases/satellite cards;
- explicit project-history timeline;
- multi-page sections like about/catalog/contacts;
- grouping sites by themes and transitions;
- CMS/JSON/Markdown content source;
- real dynamic server data;
- production form endpoint.
